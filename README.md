STEWRE
======

Super Text Editor With Regular Expressions

Editor creado para practicar con expresiones regulares en la asignatura Modelos de Computación del Grado de Informática (UGR), año 2013.

Editor muy útil para comprobar que las expresiones regulares que estás probando hacen lo que se supone que deben hacer.

No se necesita servidor web para ejecutar el editor
